# FrameZero v81u — Frases de Impacto e Versículos

Base estável: v81j — pasta simples.

## O que mudou

A versão v81u mantém a estrutura simples da v81j e adiciona arquivos extras dentro da pasta de cada corte:

```txt
Nome do Corte - 00-01-15 - score 065/
├── Nome do Corte - 16x9.mp4
├── Nome do Corte - 9x16.mp4
├── legenda.srt
├── legendas-redes-sociais.txt
├── frases-de-impacto.txt
├── versiculos.txt
├── resumo-do-corte.txt
└── metadata.json
```

## Arquivos novos por corte

### frases-de-impacto.txt
Lista frases fortes, chamativas ou compartilháveis detectadas no trecho.

### versiculos.txt
Lista referências bíblicas citadas no corte, quando existirem.

O sistema não inventa versículos. Ele apenas registra referências encontradas no texto ou retornadas pela IA, como:

- Lucas 18
- João 3:16
- Salmo 23

## Arquivos novos na pasta geral dos cortes finais

Quando os cortes finais são gerados, a pasta principal também recebe:

```txt
indice-dos-cortes.txt
frases-de-impacto-da-pregacao.txt
versiculos-da-pregacao.txt
```

## Observação

A base continua sendo v81j. As versões experimentais posteriores continuam desconsideradas.
