# Painel oficial OBS

O painel oficial do FrameZero Clips para o OBS é:

```text
https://clips.framezeroai.com.br/obs
```

Não existe painel local `app/painel.html`.

O aplicativo local continua rodando apenas como ponte técnica:

```text
ws://localhost:8765
```

No OBS, use:

```text
Docks > Custom Browser Docks
Nome: FrameZero Clips
URL: https://clips.framezeroai.com.br/obs
```
